Проанализируй текст и выведи результат в формате JSON по структуре:

{
  "emotion": "",
  "insights": {
    "emotional_dynamics": "",
    "key_triggers": [],
    "physical_reaction": "",
    "coping_strategies": {
      "effective": "",
      "ineffective": ""
    },
    "recommendations": []
  }
}

В поле emotion укажи одну из эмоций: angry, disgust, fearful, happy, neutral, sadness, surprised.

В разделе insights:

    emotional_dynamics: опиши изменение эмоционального состояния в тексте;
    key_triggers: перечисли ключевые события/факторы (массив строк);
    physical_reaction: опиши физические проявления эмоции (если упомянуты);
    coping_strategies: опиши стратегии совладания с эмоциями и событиями, укажи эффективные и неэффективные;
    recommendations: предложи 3–5 рекомендаций по улучшению эмоционального состояния и или помогающие решить проблему поднимающуюся в тексте (массив строк).

Если информации недостаточно для заполнения поля — оставь его пустым. Избегай домыслов, сохраняй нейтральный тон.
